# Gamified Environmental Platform 🌍🎮

## Overview
An interactive web application designed to promote sustainability and eco-friendly habits through gamification. This platform encourages users to track their daily environmental impact, complete green challenges, and earn rewards, transforming personal climate action into an engaging, community-driven experience.

## Key Features
* **Interactive Dashboard:** A centralized hub where users can log daily eco-friendly activities (e.g., recycling, using public transit, reducing energy consumption).
* **Gamification Engine:** A robust system that awards points, badges, and levels based on user consistency and impact.
* **Global/Local Leaderboards:** Fosters healthy competition by allowing users to compare their environmental impact scores with friends and the broader community.
* **Impact Analytics:** Visualizes the user's carbon footprint reduction over time using dynamic charts and metrics.
